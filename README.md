## Description
This is the official server for Scapetown.

#### What is scapetown?
Scapetown is one of the largest escape rooms in the world, created by secondary school students. [Learn more](https://jegaathetmaken.nl/scapetown/)

## Required

- typescript
- nodejs


## Installation

```bash
$ yarn
```
## Running the app

```bash
# development
$ yarn start:dev

# production mode
$ npm run start:prod
```
